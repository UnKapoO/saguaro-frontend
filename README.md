# 🌵 Saguaro Barefoot Chile - Frontend

Plataforma web de E-commerce y Módulo Transaccional para la marca de calzado ergonómico Saguaro Barefoot. Este repositorio contiene exclusivamente la capa de presentación (Frontend/UX) orientada a la conversión y experiencia de usuario.

Proyecto desarrollado como parte del MVP para la asignatura Taller Aplicado de Programación.

## 🚀 Tecnologías Utilizadas

* **Librería Principal:** React
* **Lenguaje:** TypeScript
* **Estilos:** Tailwind CSS
* **Despliegue:** Vercel (CI/CD)

## 📦 Funcionalidades del MVP (Capa Cliente)

* **Catálogo Dinámico:** Visualización de productos, tallas y stock en tiempo real.
* **Carrito de Compras:** Gestión de estado global para los productos seleccionados.
* **Integración de Pagos:** Flujo de checkout preparado para conexión con API de Transbank (Webpay).
* **Dashboard Administrativo:** Interfaz gráfica para la visualización de métricas de Inteligencia de Negocios (BI) como Ticket Promedio y alertas de Stock Crítico.

## ⚙️ Instalación Local

Para correr este proyecto en tu entorno local:

1. Clona este repositorio: `git clone https://github.com/tu-usuario/saguaro-frontend.git`
2. Instala las dependencias: `npm install`
3. Inicia el servidor de desarrollo: `npm run dev`
